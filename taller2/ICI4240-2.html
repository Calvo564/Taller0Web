Ingenieria Web

Informacion sobre el Curso.
Sigla: ICI4240-2
En este curso los participantes desarrollarán las habilidades necesarias para aplicar un modelo de ingeniería web y crear una aplicación web.
Además, conocerán cada una de sus fases, los diagramas que se deben desarrollar y la manera de implementar la aplicación web.
Valor Academico: 3 Creditos.
